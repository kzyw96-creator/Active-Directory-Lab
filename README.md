# Active Directory Lab - Windows Server 2022

This repository contains my hands-on Active Directory lab built using Windows Server 2022 in VMware Workstation.

## Lab Overview
- Installed Active Directory Domain Services (AD DS)
- Configured DNS Server
- Assigned Static IP Address
- Created Organizational Units (HR, IT, Sales)
- Created Users and Security Groups
- Configured Group Policy Objects (GPO)
- Verified network connectivity using ipconfig and ping

## Environment
- Windows Server 2022
- VMware Workstation
- Active Directory
- DNS
- Group Policy Management
